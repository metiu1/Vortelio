Everything you need to run AI locally

Ollama compatible

Works with any Ollama model. Download, manage and run models directly from the built-in UI. Zero extra setup.

Multimodal

Vision models, image generation, audio transcription and speech synthesis — all running locally on your GPU.

AI Agents

Install and run autonomous AI agents. Each agent runs in an isolated environment with its own tools and memory.
